Ouvintes mensais é o número mais fácil de inflar no streaming. Uma playlist editorial, um push de gravadora, um mês de tráfego pago — e o número explode. Some o impulso, some o número. Não é fã. É passageiro.

O BR TOP 100 usa o Pulso do Artista — um índice composto que cruza três sinais para medir base de fãs genuína e presença ativa na cena:

Pulso = (Popularidade × 1.000) + (Seguidores ÷ 1.000) + (Pop. top track × 10) // Popularidade — índice de 0 a 100 do Spotify que mede quão tocado o artista // está AGORA. Sobe com streams recentes, saves e playlists; cai quando esfria. // É o termômetro de relevância do momento, não o histórico acumulado. // Seguidores — base acumulada no Spotify. Fã que escolheu seguir. Difícil de manipular. // Top track — popularidade da faixa mais quente. O artista tem hit circulando agora?

Dois artistas com a mesma popularidade bruta ocupam posições diferentes conforme base de fãs e hit atual. Sem empates. Com movimento real a cada atualização.

Sobre o BR TOP 100

O BR TOP 100 nasceu de uma inquietação: a cena eletrônica brasileira é uma das mais fortes do mundo, mas nunca teve um termômetro honesto de quem realmente importa — não um ranking de quem paga mais playlist, mas de quem tem fã de verdade.
